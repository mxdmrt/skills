# Agent Skills

A small collection of agent skills for sharpening product thinking before implementation.

Each skill lives in its own directory and can be installed individually.

## Discovery & UX

These skills help pressure-test ideas before they turn into designs, specs, or code.

- **challenge-me** — Stress-test a product idea, user story, or UX direction with direct questions about the problem, evidence, tradeoffs, risks, accessibility, and what should be validated first.

```bash
npx skills@latest add mxdmrt/skills/challenge-me
```

## Repository Structure

- `challenge-me/` — Skill definition and instructions
